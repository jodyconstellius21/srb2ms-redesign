Hosting? Read the rules

1. No mandatory music wad downloads above 20MB

If you would like to host a larger music WAD, make sure it isn't a mandatory download so players aren't required to have it to join your server.

2. No advertising in server titles

Exceptions to this rule include WAD hosting services, and domain names that point to the server. A WAD hosting service should ideally be a file index, and must contain only the files that the server is actually using.

3. No inappropriate content in server titles or addons

This includes slurs, links to pirated content or pornography, or other content generally deemed inappropriate. Also, server titles should not contain swearing.

4. No netgame hacking

By hacking we specifically mean:

  • Using illegitimate EXEs to cheat
  • Forcing clients or servers to crash using certain add-ons or exploits
  • Hosting add-ons that modify clients' settings and/or interfere with the vanilla game even after restarting

5. No netgame harassment

Harassment includes but is not limited to:

  • Spamming
  • Flaming
  • Sexual conversations

6. No hosting unauthorized ports or edits of released content

Respect the artists in our community by keeping such files to yourself.
